# Gaja Auction NPL Consulting - Design System

## 1. Design Concept (디자인 컨셉)

- **Theme**: Ultra Premium Deep Navy & Champagne Gold
- **Target Emotion**: Trust(신뢰), Luxury(고급), Heritage(전통), Innovation(혁신)
- **Visual Style**: Modern Glassmorphism 2.0, Minimalist Typography, Abstract Geometric Backgrounds

## 2. Color Palette (색상 규정)

| Color Name | Hex Code | Usage |
| :--- | :--- | :--- |
| **Primary Navy** | `#050b14` | Main Background (Global) |
| **Secondary Navy** | `#0a192f` | Section Backgrounds |
| **Champagne Gold** | `#D4AF37` | Primary Accents, Borders |
| **Gold Gradient** | `linear-gradient(135deg, #D4AF37, #F5DEB3, #D4AF37)` | CTAs, Headlines, Icons |
| **Text White** | `#ffffff` | Primary Headings |
| **Text Off-White** | `#e6f1ff` | Body Text |
| **Text Gray** | `#a8b2d1` | Subtitles, Secondary Info |
| **Glass BG** | `rgba(255, 255, 255, 0.03)` | Cards (Glassmorphism) |

## 3. Typography (타이포그래피)

- **Headings**: `Playfair Display` (Serif)
  - Usage: H1, H2, H3, Brand Logo
  - Characteristics: Elegant, High Contrast, Wide Letter Spacing
- **Body**: `Noto Sans KR` (Sans-serif)
  - Usage: Paragraphs, UI Elements
  - Characteristics: Clean, Legible, Modern

## 4. UI Components (컴포넌트)

### Buttons

- **Primary Pill**:
  - Shape: Rounded Pill (`50px`)
  - Background: Gold Gradient
  - Shadow: Soft Gold Glow
- **Outline Pill**:
  - Shape: Rounded Pill
  - Border: 1px Solid Gold
  - Background: Transparent

### Glass Cards 2.0

- **Background**: Ultra-thin transparency (`0.03`)
- **Border**: Subtle white inset (`0.08`)
- **Hover**:
  - Transform: Lift Up (`-15px`)
  - Accent: Gold Top Border Reveal (`scaleX`)

## 5. Layout & Spacing

- **Container**: Max-width `1300px` (Wide)
- **Spacing**: `120px` Vertical Padding (Generous negative space)
- **Grid**: 3-Column Standard (`repeat(3, 1fr)`)

## 6. Icons & Imagery

- **Icons**: Font Awesome (Gold Gradient Color)
- **Imagery**: Abstract radial gradients, Dark cinematic overlays
