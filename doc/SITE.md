# Gaja Auction NPL Consulting Website Vision

## 1. Vision

To build a high-end, trustworthy, and authoritative website for "Gaja Auction NPL Consulting". The site should blend the professional reliability of a law firm with the innovative edge of an AI tech company.

- **Key Emotions**: Trust, Authority, Innovation, Premium.
- **Color Palette**: Deep Navy (#0a192f), Gold (#ffd700), White/Light Gray.
- **Typography**: Clean, modern sans-serif (e.g., Inter, Roboto) for body; Serif (e.g., Playfair Display) for headings if appropriate for "premium" feel, or sleek sans-serif.

## 2. Target Audience

- Individual investors looking for NPL (Non-Performing Loan) opportunities.
- Corporate clients seeking auction consulting.
- Real estate developers needing AI-backed market analysis.

## 3. Core Features

- AI-powered Auction Analysis (highlighted in copy).
- NPL Investment Consulting explanation.
- Searchable Auction Listings (mockup for now).
- Professional Company Introduction.
